# Dataset Information

## Primary Dataset
Name    : Sentiment Analysis for Mental Health
Source  : Kaggle
Link    : https://www.kaggle.com/datasets/suchintikasarkar/sentiment-analysis-for-mental-health
Size    : 53,043 samples
Format  : CSV (Combined Data.csv)
Columns : statement, status

## Labels
0 → Anxiety
1 → Bipolar
2 → Depression
3 → Normal
4 → Personality Disorder
5 → Stress
6 → Suicidal

## Secondary Dataset (Real Clinical Testing)
Name    : MentalChat16K
Source  : HuggingFace — ShenLab
Link    : https://huggingface.co/datasets/
          ShenLab/MentalChat16K
Files   : Interview_Data_6K.csv
          Synthetic_Data_10K.csv
Used for: Real-world inference testing only
